Click on me to get to scRNA-seq workflow
  1. Obtain Barcoded cDNA
    • - Generate barcoded cDNA using the 10x Genomics platform.
    • - Prepare the cDNA using the Single Cell 3’ or 5’ Gene Expression kits.
    • - Use Visium Spatial Gene Expression kits for spatial transcriptomics.
  2. Amplify and Purify cDNA
    • - Perform PCR amplification using biotinylated primers.
    • - Remove unbarcoded cDNA using streptavidin pulldown.
    • - Ensure efficient depletion of artifacts for high-quality sequencing.
  3. Prepare Sequencing Library
    • - Use the Oxford Nanopore Ligation Sequencing Kit.
    • - Complete library preparation within approximately three hours.
  4. Sequence on PromethION Platform
    • - Load the library onto a PromethION Flow Cell.
    • - Generate ~80M full-length, cell-assigned reads per flow cell.
    • - Use PromethION platforms for flexible throughput options.
  5. Analyze Data
    • - Use the EPI2ME wf-single-cell pipeline for analysis.
    • - Obtain gene and isoform count matrices, UMAP plots, and BAM files.
    • - Leverage cloud or local compute options for workflow execution.
